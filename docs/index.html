Open-Web-Console

（一个类似浏览器控制台Console的插件，可以实现在页面中进行打印。）

引入

首先你需要通过命令行安装，就像这样： 

npm install --save open-web-console

安装好了以后，引入即可： 

import OpenWebConsole from 'open-web-console';

当然，你也可以通过CDN的方式引入： 

<script src='https://cdn.jsdelivr.net/npm/open-web-console@0'></script>

语法

首先，你需要通过new创建打印对象： 

var myConsole = new OpenWebConsole({
    el: DOM
});

然后，直接调用上面的打印方法即可，可用的打印方法有：log、trace、info、error、warn、debug。

例子

下面就是一个运行的例子：

具体代码如下： 

var myConsole = new OpenWebConsole({
    el: document.getElementById('console')
});

myConsole.error('Open-Web-Console');
myConsole.log([1, 2, 3]);
myConsole.debug(window);
